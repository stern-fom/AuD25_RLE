# Praktische Übung: Algorithmen & Datenstrukturen

Dieses Repository ist Teil einer praktischen Übung im Kurs **"Algorithmen & Datenstrukturen"**.

---

## Inhalt

In diesem Projekt werden grundlegende Algorithmen und Datenstrukturen implementiert und analysiert. Ziel ist es, die theoretischen Konzepte aus der Vorlesung praktisch anzuwenden und zu vertiefen.

---

## Aufbau

- **Code:** Enthält Implementierungen der Algorithmen, z.B. Kompressionsverfahren wie Run-Length Encoding (RLE).
- **Dokumentation:** Kommentare im Code erläutern die Funktionsweise der Algorithmen.
- **Beispiele:** Testfälle und Beispielaufrufe zur Veranschaulichung der Implementierungen.

---

## Installation

Das Projekt benötigt keine speziellen Abhängigkeiten und kann direkt mit Python 3 ausgeführt werden.

```bash
python main.py
